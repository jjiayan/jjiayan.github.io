## 풀이

```python
from sys import stdin

n, k = map(int, stdin.readline().split())
cnt = 0

arr = [True] * (n+1)

for i in range(2, n+1):
	for j in range(i, n+1, i):
		if arr[j]:
			arr[j] = False
			cnt += 1
			if cnt == k:
				print(j)
				break
```
